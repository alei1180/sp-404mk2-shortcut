Комбинации с кнопкой [SHIFT]
Удерживая [SHIFT]ФункцияОписание
Pad [1]FIXED VELOCITYФиксирует в максимальном значении 127 силу нажатия (громкость) пэда.
Pad [2]16 VELOCITYПошагово изменеяет силу нажатия пэда (громкости).
Pad [3]CUEВыводит звук только через разъем наушников PHONES и не выводит на разъем LINE OUT или USB. Таким образом при игре вживую или в подобных ситуациях можно предварительно прослушать (проверить) звук сэмпла в наушниках. Нельзя отправлять два или более сэмпла в CUE одновременно.
Pad [4]CHROMATICВоспроизводит сэмплы изменяя их высоту (питч) в хроматической гамме с помощью пэдов. Хроматический режим нельзя использовать, когда горит кнопка PATTERN SELECT. В хроматическом режиме может воспроизводиться только один сэмпл за раз.
Pad [5]EXCHANGEМеняет местами сэмплы между двумя выбранными пэдами или банками.
Pad [6]INIT PARAMСбрасывает параметры, заданные в сэмпле (например, начальную точку, конечную точку, темп и другие) до значений по умолчанию.
Pad [7]PAD LINKОбъединяет несколько пэдов (до четырех) в одну группу. Задав группу, вы можете заставить все пэды в группе воспроизводить звук, просто нажав один пэд из этой группы. Можно создавать до 10 групп.
Pad [8]MUTE GROUPГруппирует сэмплы, которые вы не хотите воспроизводить одновременно (не хотите накладывать друг на друга). При попытке одновременного воспроизведения всех сэмплов в этой группе будет слышен только тот сэмпл, который начал воспроизводиться последним. Группа может состоять из 16 сэмплов. Можно создавать до 10 групп.
Pad [9]METRONOMEВлючает/отключает звук метронома. Метроном - это встроенный инструмент для отсчета ритма и темпа.
Pad [10]COUNT-INВключает отсчет времени перед началом сэмплирования. COUNTIN 1MEAS - однократный отсчет. COUNTIN 2MEAS - двукратный отсчет. COUNTIN WAIT - когда уровень входного аудиосигнала превышает значение, установленное в параметре Auto Trig Level, начинается сэмплирование. COUNTIN OFF - отсчет времени не используется, сэмплирование начинается сразу после нажатия пэдов [1]–[16], а затем нажатия кнопки [REC].
Pad [11]TAP TEMPOИнтуитивно устанавливает темп, при нажимании на пэд в такт, как если бы вы хлопали в ладоши в такт музыке. Это полезная функция, когда вы хотите играть в такт с другими инструментами или в ритме композиции.
Pad [12]GAINОтображает меню настроек усиления UTILITY MENU > SYSTEM > GAIN.
Pad [13]UTILITYОтображает меню системных настроек UTILITY MENU.
Pad [14]IMPORT/EXPORTОтображает меню импорта/экспорта UTILITY MENU > IMPORT (IMPORT/EXPORT MENU).
Pad [15]PAD SETTINGОтображает меню настроек пэдов (UTILITY MENU > PAD SET).
Pad [16]EFX SETTINGОтображает меню настроек эффектов (UTILITY MENU > EFX SET).
[BUS FX] buttonMUTE BUSВременно отключает звук, передаваемый на шину (звук воспроизведения сэмпла или звук, поступающий на входной разъем INPUT), и выводит только звук эффекта.
[HOLD] buttonPAUSEПриостановливает воспроизведение текущих сэмплов.
[EXT SOURCE] buttonINPUT SETTINGОтображает экран настроек входа.
[SUB PAD] buttonPROJECTОтображает экран настроек входного сигнала (SELECT PROJECT).
[MARK] button (не менее 3 сек)SAVE EFX PARAMСохраняет основные параметры эффектов, назначенных на шины BUS 1 и BUS 2.
[PITCH/SPEED] buttonENVELOPEУстанавливает изменение громкости во время воспроизведения сэмпла.
[START/END] buttonCHOPРазделяет сэмпл по позициям маркеров и назначает полученные сэмплы на отдельные пэды.
[PATTERN SELECT] buttonUNDOОтменяет только что введенные (записанные) данные.
* Работает только при записи паттерна.
[ROLL] buttonROLL SETПоказывает, с какой частотой повторяется сэмпл в режиме ROLL (скорость повторения).
[REVERSE] button + pads [1]-[16]Pad MUTEВключает/выключает (mute) звук выбранного пэда.
[REVERSE] button + [REMAIN] buttonPad MUTE MODEФиксирует кнопки [SHIFT] и [REVERSE] в нажатом состоянии, для более удобного включения/выключения (mute) пэдов.
[LOOP] buttonPING-PONG LOOPЗацикливает сэмпл, воспроизводя его попеременно вперед и назад.
[GATE] buttonGATE ALL ON/OFFВключает/выключает параметр GATE для всех сэмплов в выбранной банке.
[BPM SYNC] buttonSYNC ALL ON/OFFВключает/выключает параметр BPM SYNC для всех сэмплов в выбранной банке.
[REC] buttonTR-RECПереключает в режим записи паттерна с помощью TR-REC. Записывает сэмплы в играющий паттерн.
* Работает только при воспроизведении паттерна.
Bank [A/F]-[E/J] buttonsBANK VOLUMEРегулирует общую громкость для указанного банка.
[REMAIN] button (не менее 3 сек)LIVE MODEОтключает кнопки, не используеющиеся при живом выступлении (связанных с сэмплированием и редактированием).
[COPY] buttonBANK PROTECTЗащищает сэмплы и паттерны в банке от копирования, перезаписи или случайного удаления.
[VALUE] knob (нажать)MARKПереключает функцию, вызываемую кнопкой [MARK].
* Переключение между SBS (skip-back sampling) и Looper.
[RESAMPLE] buttonSAMPLE MERGE MODEПереключает в режим объединения сэмплов (Sample Merge).
[RECORD SETTING] buttonSOUND GENERATOR MODEПереключает в режим генератора звука (часто применяется для генерации баса).
[EXIT] buttonSTOPОстанавливает воспроизведение всех сэмплов.
Комбинации с кнопкой [REMAIN]
Удерживая [REMAIN]Функция (описание)
Pads [1]-[16]Устанавливает, шину эффектов для выбранного пэда (BUS 1, BUS 2, Без шины).
Bank [A/F]-[E/J] buttonsУстанавливает шину эффектов для всех сэмплов в банке (BUS 1, BUS 2, Без шины).
[MFX] buttonОставляет открытым экран редактирования эффектов (по умолчанию экран открыт несколько секунд).
[BUS FX] buttonМеняет местами эффекты BUS 1 и BUS 2. Значения параметров сохраняются.
Edits the following parameters on the pitch/speed settings screen
  • SPEED
  • PITCH
  • VOLUME
  • PAN
  • BPM
  • BPM SYNC
  • GATE
  • LOOP
  • REVERSE
Одновременное редактирование параметров сэмплов Pitch/Speed в одной банке: SPEED, PITCH, VOLUME, PAN, BPM, BPM SYNC, GATE, LOOP, REVERSE.
Edits the following parameters on the envelope settings screen
  • ATTACK
  • HOLD
  • RELEASE
  • BPM SYNC
  • GATE
  • LOOP
  • REVERSE
Одновременное редактирование параметров сэмплов Envelope в одной банке: ATTACK, HOLD, RELEASE, BPM SYNC, GATE, LOOP, REVERSE.
Комбинации с кнопкой [VALUE]
Удерживая [VALUE]Функция (описание)
Pads [1]-[16]Выбирает сэмпл без воспроизведения звука. Также выбирает паттерн, если горит [PATTERN SELECT] (паттерн не играет).
[BUS FX] buttonВременно отключает эффекты на шинах BUS 3 и BUS 4.
[SUB PAD] buttonМинимизирует небольшие расхождения во времени при игре нескольких сэмплов одновременно.
Effect buttons ([FILTER+DRIVE] - [MFX])Применяет эффекты только пока удерживается кнопка эффекта.
[GATE] buttonВключает/выключает режим воспроизведения one-shot playback для сэмпла.
Комбинации с кнопкой [DEL]
Удерживая [DEL]Функция (описание)
[MARK] buttonУдаляет из паттерна автоматизацию с эффектами, записанные с помощью EFX MOTION REC.
* Доступно на экране редактирования паттерна.
[REVERSE] buttonУдаляет из паттерна автоматизацию глушения пэдов, записанные с помощью EFX MOTION REC.
* Доступно на экране редактирования паттерна.
[EXIT] buttonУдаляет все сэмплы или паттерны в банке.

Редактирование EnvelopeОдновременное редактирование параметров сэмплов Envelope в одной Mute Group: ATTACK, HOLD, RELEASE, BPM SYNC, GATE, LOOP, REVERSE.

Горячие клавиши в DJ mode
Действие [DJ mode]ПараметрОписание
[SHIFT] + [REVERSE] + pad [13] (CH1)
[SHIFT] + [REVERSE] + pad [15] (CH2)		MUTEЗаглушает воспроизводящийся сэмпл.
[SHIFT] + pads [1]–[16]-Воспроизведение сэмпла с позиции маркера, установленного для этого сэмпла.
[SHIFT] + [REMAIN]-Фиксирует кнопку [SHIFT] в нажатом состоянии. Упрощает выбор маркера и воспроизведение. Нажмите [EXIT] для отмены.
[SHIFT] + [MARK]-Добавление маркеров во время проигрывания сэмпла в режиме DJ.
[SHIFT] + [START/END]-Редактирование маркеров во время проигрывания сэмпла в режиме DJ.
[SHIFT] + [DEL] + pads [1]–[16]-В режиме DJ удаляет маркеры, установленные для сэмплов.
[SHIFT] + [ROLL]ROLL SIZEУстанавливает интервал ролла (1/4, 1/2, 1 или 2 такта). Устанавливайте до начала воспроизведения ролла.
[ROLL] + pad [13] (CH1)-Зацикливает воспроизведение сэмпла с заданным интервалом (ROLL). Не работает, если интервал ролла длиннее сэмпла.
[ROLL] + pads [1]–[4]-Изменяет интервал ролла во время его воспроизведения.
Pad [1]: четверть (1/4)
Pad [2]: половинная (1/2)
Pad [3]: целая нота (1)
Pad [4]: две целых (2)
[REMAIN] + pad [14] (CH1)
[REMAIN] + pad [16] (CH2)		BUS FXУстанавливает шину для сэмплов CH1/CH2. Нажатие pad [14] или [16] переключает: BUS-1 → BUS-2 → DRY.
[RESAMPLE] + [VALUE] knobVOLUME CURVEВыбирает характеристику кривой громкости для слайдеров (CH1 LEVEL, CH2 LEVEL, X-FADE) в режиме DJ. Переключение: FAST CUT → LINEAR → SQUARE → CUBIC.
[DEL] + pad [2] + pad [6] (CH1)
[DEL] + pad [4] + pad [8] (CH2)		BPMСбрасывает темп к значению по умолчанию.
[START/END] button-Переключает функцию ручки [CTRL 3] между CUE MIX и X-FADE.
[PITCH/SPEED] button-Изменяет отображение цифр BPM (целые → сотые → десятые → целые).
[MARK] button-Переключает вид экрана между EFX и MIXER.
[BPM SYNC] button-Выбирает канал (CH1/CH2) для управления реверсом (кнопка [REVERSE]).
Горячие клавиши в TR-REC
Действие [TR-REC]Описание
[DEL] + [A/F] buttonУдаляет ноты (на один такт), соответствующие выбранному пэду.
[DEL] + [B/G] buttonУдаляет ноты (на один такт) для всех пэдов.
[ROLL] + [CTRL 1] knobЗаписывает движение ручки [CTRL 1] в шаги (активно в режиме "TRIG").
[ROLL] + [CTRL 2] knobЗаписывает движение ручки [CTRL 2] в шаги (активно в режиме "TRIG").
[ROLL] + [CTRL 3] knobЗаписывает движение ручки [CTRL 3] в шаги (активно в режиме "TRIG").
[VALUE] knob (press) + [SUB PAD] + pads [1]-[16]Выбирает сэмплы без их воспроизведения.
